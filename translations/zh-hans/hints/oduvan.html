你能用这个特殊的任务来解释函数吗？

与大乐。这就是我们在CheckiO上有提示的原因

想象一下，你需要将两个数字相乘并打印（输出）结果，这样你就可以在控制台中看到它。例如，5和6。

print(5*6)

输出：

30

现在我们可以用变量来代替5和6。为了什么？你会在后面的例子中看到。

a = 5
b = 6
print(a*b)

输出：

30

你可以改变变量a，再次运行脚本，并看到不同的结果。

a = 6
b = 6
print(a*b)

输出：

36

或者你可以把你的代码复制两次，改变变量，看到两次乘法的结果。

a = 5
b = 6
print(a*b)


a = 6
b = 6
print(a*b)

输出结果：

30
36

现在我可以告诉你函数是如何简化上述代码的。

def mult_two(a, b):

    return a*b

print(mult_two(5, 6))
print(mult_two(6, 6))

输出结果：

30
36

正如你所看到的，函数mult_two ，现在掌握了所有的逻辑，你只需改变参数就能看到不同的结果。

当然可以。这里有一张图片，显示了函数结构：

这里又是我们的函数：

def mult_two(a, b):

    return a*b

总是以def 开始。

函数的名称mult_two - 你将使用该名称在将来调用该函数

两个参数(a, b) - 这些是变量，当函数被调用时将得到它们的值。例如，当我们调用函数mult_two(5, 6) - 在函数内部，我们将变量a设置为5，变量b设置为6。

你还可以看到带有类型提示的函数定义。

def mult_two(a: int, b: int) -> int:

    return a*b

不要被: int 和-> int 所迷惑。它们向用户（阅读代码的人）解释了参数的预期类型和函数调用结果的类型。但现在可以忽略这些。

现在，以四个空格开始的函数块和return ，说明了函数调用的结果应该是什么 return a*b 。

在我们的例子中，print(mult_two(5, 6)) - 两个值5和6的乘法结果将被返回并作为函数print 的第一个参数。

如果你想让你的函数中有多于一行的代码--所有的行都应该有前4个空格。例如，让我们再使用一个变量c，在这里我们放置乘法的结果并返回结果

def mult_two(a, b):

    c = a*b
    
    return c
