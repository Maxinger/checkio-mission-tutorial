荣幸之至。这就是我们在CheckiO上有提示的原因!

想象一下，你需要将两个数字相乘，并打印（输出）结果，这样你就可以在控制台看到它。例如，5 和6 。

console.log(5*6);

输出：

30

现在，我们可以使用变量来代替5 和6 。为了什么？你会在后面的例子中看到。

let a = 5;
let b = 6;
console.log(a*b);

输出：

30

你可以改变变量a ，再次运行脚本，并看到不同的结果。

let a = 6;
let b = 6;
console.log(a*b);

输出：

36

或者你可以把你的代码复制两次（在复制过程中没有let ），改变变量，看看两次乘法的结果。

let a = 5;
let b = 6;
console.log(a*b);


a = 6;
b = 6;
console.log(a*b);

输出：

30
36

现在我可以告诉你函数是如何简化上述代码的。

function multTwo(a, b) {

    return a*b;
}

console.log(multTwo(5, 6));
console.log(multTwo(6, 6));

输出：

30
36

正如你所看到的，函数multTwo ，现在掌握了所有的逻辑，你只需改变参数就能看到不同的结果。

当然可以。以下是该函数再次：

function multTwo(a, b) {

    return a*b;
}

总是以function 开始。

函数的名称multTwo - 你将使用该名称在将来调用该函数。

两个参数(a, b) - 这些是变量，当函数被调用时将得到它们的值。 它们的值，当函数被调用时。例如，当我们调用函数 multTwo(5, 6) - 内，我们将变量a 设置为5 ，将变量b 设置为6 。

你还可以看到带有类型提示的函数定义。

function multTwo(a: number, b: number): number {

    return a*b;
}

不要被: number 所迷惑。它向用户（阅读代码的人）解释了参数的预期类型和函数调用结果的类型。但现在忽略这些也是可以的。

现在，{ 、} 和return 里面的函数块说明了函数调用的结果应该是 return a*b 。

在我们的例子中，console.log(multTwo(5, 6)) - 两个值相乘的结果5 和6 将被返回并作为函数console.log() 的第一个参数。

你的函数可以有多于一行的代码。例如，让我们再使用一个变量c ，在这里我们放置乘法的结果并返回结果。

function multTwo(a, b) {

    let c = a*b;
    
    return c;
}
